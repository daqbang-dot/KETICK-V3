‚Äčūüõ†ÔłŹ Troubleshooting & Soalan Lazim (FAQ)
‚Äč1. Mengapa data saya hilang selepas menutup pelayar?
‚ÄčPunca: Anda mungkin menggunakan mod Incognito/Private atau pelayar anda ditetapkan untuk memadam cache secara automatik.
‚ÄčPenyelesaian: Gunakan mod pelayar biasa. Pastikan anda menggunakan butang EXPORT BACKUP sebelum menutup pelayar atau membersihkan sejarah (history).
‚Äč2. Gambar resit atau logo tidak muncul selepas Import.
‚ÄčPunca: Jika fail JSON terlalu besar, sesetengah pelayar mengehadkan saiz imej dalam Local Storage.
‚ÄčPenyelesaian: Gunakan gambar yang telah dikecilkan saiznya (compressed) sebelum dimuat naik ke dalam sistem untuk prestasi optimum.
‚Äč3. Butang WhatsApp tidak berfungsi.
‚ÄčPunca: Penyekat iklan (Ad-blocker) atau tetingkap timbul (Pop-up blocker) mungkin menghalang pautan luar.
‚ÄčPenyelesaian: Benarkan (Allow) pop-up untuk laman BizPro ini dalam tetapan pelayar anda.
