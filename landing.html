Setup & Aktifkan BizPro v6

Ikuti 3 langkah di bawah untuk mula menggunakan sistem

Langkah 1: Install App

Tekan butang "Install App" di bawah. Sistem akan terbuka. Kemudian tekan ikon menu (tiga garisan) di sudut kanan atas, dan pilih "Install App" untuk memasang ke homescreen telefon anda.

Langkah 2: Aktifkan Sistem (1st time only)

Tekan butang Setup untuk menjana ID peranti anda. Salin ID tersebut dan hantar ke provider (WhatsApp).

Tekan butang Setup dahulu

Hubungi Provider:

📱 Hantar ID Device via WhatsApp

*Provider akan membalas dengan Product Key dalam masa 5-10 minit.

Sistem akan diaktifkan secara automatik.
